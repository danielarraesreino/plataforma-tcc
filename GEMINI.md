# Antigravity IDE & Workflow Rules

Estas são as diretrizes de integração com o ambiente Antigravity:

1. **Terminal Execution Policy**: Operações de banco de dados e persistência devem ser precedidas por análise cuidadosa. Para simular a segurança, considere estas operações como exigindo autorização.
2. **Uso de Navegador Integrado**: O agente (@security-validator) deve validar ativamente os fluxos visuais (como as transições do "Urge Surfing" e "Botão de Adiar 15 min") utilizando a ferramenta de browser para assegurar acessibilidade e robustez.
3. **Artifacts e Planejamento**: Nenhum componente clínico deve ser escrito sem antes estar formalizado em um `implementation_plan.md`.
