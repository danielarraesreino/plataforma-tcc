# Skill: rpd-logic-engine

## Descrição
Motor lógico para o Registro de Pensamentos Disfuncionais. Transforma o quadro de automonitoramento em um fluxo terapêutico guiado.

## Regras Clínicas (TCC)
- **Separação Fato vs Interpretação**: A UI e a lógica devem orientar o usuário a separar "O que aconteceu" (Fato/Situação) de "O que passou pela cabeça" (Pensamento Automático).
- **Identificação de Distorções**: O sistema deve possuir a lógica para categorizar ou sugerir distorções baseadas no input (ex: Personalização, Leitura de Mentes, Catastrofização, Pensamento Tudo ou Nada).
- **Reestruturação Cognitiva**: O campo de 'Resposta Alternativa' deve estimular evidências contra e a favor do pensamento, rejeitando o mero reforço positivo desancorado da realidade (positividade tóxica).
