# Skill: tcc-component-scaffold

## Descrição
Template e diretrizes para geração de componentes UI em React para a plataforma.

## Regras de Design
- **Estética Premium**: A interface deve ser "Wow", transmitindo acolhimento e profissionalismo.
- **Psicologia das Cores**: Evite o uso de vermelho puro ou cores alarmantes. Use tons pastéis, azuis suaves ou neutros para criar uma atmosfera calma.
- **Glassmorphism e Micro-animações**: Adicione feedback sutil de interações. Um formulário de RPD que "respira" junto com o paciente.
- **Vanilla CSS**: Os componentes devem importar e encapsular seu próprio CSS puro sem depender de frameworks como Tailwind (a menos que explicitamente solicitado).
