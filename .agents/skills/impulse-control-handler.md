# Skill: impulse-control-handler

## Descrição
Módulo comportamental para Gerenciamento e Modificação de Tendências (GMT).

## Regras e Técnicas
- **Urge Surfing (Surfar na Onda)**: Implementar interfaces visuais que auxiliem o usuário a observar o impulso sem agir sobre ele, utilizando indicadores visuais de "ondas".
- **Botão de Emergência / Timer**: Implementar um mecanismo de "adiar por 15 min". A lógica deve congelar a ação impulsiva temporariamente.
- **Cartões de Enfrentamento**: Sistema para exibir rapidamente frases ou estratégias previamente cadastradas pelo usuário quando a intensidade do impulso (0-10) for alta.
